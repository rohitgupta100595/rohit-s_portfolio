# [Project 1: Bike Sales Analysis](https://github.com/rohitgupta100595/Bike-Sales-Analysis/tree/main)

A data analysis project aimed at uncovering customer purchasing behavior and sales patterns in the Bike Dekho dataset using Microsoft Excel.

## Project Overview
This project demonstrates how Excel can be used for:
- Data cleaning & preparation
- Exploratory Data Analysis (EDA)
- Building interactive dashboards
- Deriving actionable business insights

## Objective
- Clean and transform raw bike sales data
- Analyze customer demographics, lifestyle factors, and purchase behavior
- Identify key drivers influencing bike purchases
- Build a dashboard for quick decision-making

## Tools & Skills Used
- Microsoft Excel
- PivotTables & PivotCharts
- Conditional formatting
- Calculated columns
- Data Cleaning
- EDA
- Dashboard Design
- Business Insights & Storytelling

## Key Tasks Performed
1. Data Cleaning & Preprocessing
- Removed duplicates
- Standardized column formats
- Created calculated columns like Age Group, Income Group, etc.

2. Exploratory Data Analysis
Used PivotTables to study:
- Gender, marital status, income vs. purchases
- Home ownership, cars owned vs. purchases
- Commute distance, children count vs. buying behavior

3. Dashboard Creation
![](https://raw.githubusercontent.com/rohitgupta100595/Bike-Sales-Analysis/main/0.jpg)

## Insights Summary
1. Demographics & Buyer Behavior
- Single females make the highest bike purchases
- Married females purchase the least
- Single users buy more bikes than married users
- Average-income adults are the highest buyers

2. Lifestyle & Ownership Insights
- Homeowners without cars buy the most bikes
- Purchases decline as car ownership increases
- People with 0–1 children show higher purchases

3. Commute Distance
- Users with 0–1 mile commute are the highest buyers
- Very low interest for long distance (>10 miles) commutes

4. Geography & Occupation
- North America leads in purchases
- Professionals (especially bachelors) buy the most bikes

## Conclusion
The bike market is primarily driven by:
- Urban, average-income, single professionals
- Short-distance commuters
- Homeowners with limited car ownership
- Female and bachelor demographics
These groups offer strong opportunities for targeted marketing and product strategy.
